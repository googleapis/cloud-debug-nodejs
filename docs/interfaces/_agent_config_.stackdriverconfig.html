Inherited from AuthenticationConfig.projectId

The projectId of the Google Cloud Platform project whose applications will be debugged. If not specified, we will try to use detect this through either the GCLOUD_PROJECT environment variable, through the embedded projectId in the provided key, the metadata service, or the default project configured in the gcloud cli. If none of these mechanisms are successful, the debug agent will fail to function.
